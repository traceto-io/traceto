4. Data Storage and Security

All data is stored on servers in Frankfurt, Germany (EU). All transmissions are encrypted via TLS 1.3. Data at rest is encrypted with AES-256.

HTTP payload data is scrubbed before transmission by automatic PII filtering (passwords, tokens, credit card numbers, email addresses, etc. are replaced with placeholders). Raw payloads are not stored permanently.
